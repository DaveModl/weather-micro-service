# 天气预报微服务

## 模块说明
* weather-collection-micro-service --- 天气数据采集微服务

* weather-data-micro-service --- 天气数据微服务


* weather-report-micro-service --- 天气数据API微服务


* weather-eureka-client-gateway --- 网关


* weather-eureka --- 注册中心
